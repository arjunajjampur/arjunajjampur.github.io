THEA RENDER

Overview

Altair acquired Thea render in 2014 giving it access to industry leading visualization engine. Thea render is an award winning rendering and visualization product used by millions of designers.

One of the convergences was to bring in the capabilities of Thea engine into Inspire Studio, our 3D modeling product.

My contribution

I was tasked with redesigning the post processing interface within Inspire Studio.

The goal was to make it easy to compare, re-condition and animate frames on rendered images

Inconsistencies

The initial implementation done after acquisition did not allow for enough user experience research. This led to a less than desired state.

I started by taking a closer look at the UI and noting down these Inconsistencies.

  1. The "Darkroom" UI was nothing like our design system Unity. It did not feel like it was part of Altiar.
  2. IA was sometimes illogical.
  3. Iconography was incorrect.
  4. Several post processing tools such as applying filters had a steep learning curve.

JOURNEY MAP

I then laid out the journey map for a typical post processing workflow which included several sub flows such as render now,render later, relight (a way to change lighting after image is rendered).

After testing a few layouts, I narrowed it down to a traditional post-processing style with the main image in front-left, with tools on the right and a timeline at the bottom.

A/B testing

To better understand the accessibility of rendered and in-process images, I had to redesign some image cards. These cards contained crucial information and we ran some tests to find the best solution with given set of constraints.

IMAGE GALLERY

I proposed a brand new way of pre-applying and visualizing filters on the right side of the UI

This reduced the time required to find the right filter by 70%.

RESULT

The new UI included several improvements:

  1. A new horizontal image gallery that shows both queued and completed image thumbnails.
  2. Three distinct tabs for local rendering, relight and network rendering.
  3. Post processing panel with pre-applied filter presets.
  4. 13 new icons that align with our design system guidelines.

“The image gallery filtering is a game changer for us. I also love the quick preset for novice users” -- Dir. of PM, Renderinging technologies.
